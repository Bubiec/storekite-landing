# StoreKite Landing Page

AI-powered e-commerce store builder. Built with Nuxt 3 + Tailwind CSS + @nuxtjs/i18n.

## Stack
- **Nuxt 3** — SSG/SSR
- **Tailwind CSS** — styling
- **@nuxtjs/i18n** — EN (default) + PL (`/pl/`)

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run generate
```

## Deployment
Static output to Vercel.
